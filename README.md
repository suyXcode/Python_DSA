# 📘 Data Structures and Algorithms (DSA)

A curated collection of **Data Structures and Algorithms** implementations and problem solutions for learning, practice, and interview preparation.

This repository is designed to help understand core concepts, analyze time and space complexity, and build strong problem-solving skills.

---

## 📌 Contents

- Arrays
- Strings
- Recursion
- Sorting Algorithms
- Searching Algorithms
- Linked List
- Stack & Queue
- Hashing
- Trees (BST, Binary Tree)
- Graphs (BFS, DFS, Shortest Path)
- Dynamic Programming
- Greedy Algorithms
- Backtracking
